## 📊 Netflix Content Analysis Project

This project analyzes Netflix movies and TV shows dataset to understand:
- Content distribution by type
- Release trends over years
- Ratings analysis
- Top countries producing content

## 🛠 Tools Used
- Power BI
- Python (Pandas)
- SQL
- Excel

## 📌 Key Insights
- Most content is Movies
- USA produces highest content
- Netflix content increased after 2015
